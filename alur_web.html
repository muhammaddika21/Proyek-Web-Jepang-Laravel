⚠️ Masalah Saat Ini & Solusinya
MASALAH
Halaman Budaya/Kegiatan 404 & Kepengurusan Kosong

Database MySQL di Railway masih kosong. Tabel ada, tapi isinya belum ada.
Kategori "Budaya" & "Kegiatan" tidak ditemukan → BudayaController abort 404.
Data pengurus belum diisi → kepengurusan tampil kosong.

SOLUSI
Jalankan Migrate + Seed di Railway Terminal
  • Buka Railway → klik service Laravel → tab Settings → Deploy
  • Isi Start Command: php artisan migrate --force && php artisan db:seed --force && php artisan serve --host=0.0.0.0 --port=$PORT
  • Tunggu redeploy selesai → hapus migrate/seed dari command
  • Login admin → isi data kepengurusan manual
