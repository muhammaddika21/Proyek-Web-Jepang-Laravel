```
Kamu sudah bisa baca hiragana, katakana, dan angka. Sekarang saatnya belajar sesuatu yang langsung bisa dipakai hari ini juga — salam dan kosakata dasar. Ini adalah kata-kata yang akan kamu pakai di setiap percakapan bahasa Jepang, mulai dari yang paling sederhana sampai yang kamu butuhkan saat ketemu orang Jepang pertama kali.

Artikel ini sengaja dibuat praktis: hafalkan dulu, pahami strukturnya belakangan. Grammar akan kita bahas di artikel berikutnya.


[FORMAT: Heading 2]
Salam Berdasarkan Waktu (挨拶, Aisatsu)

[FORMAT: Paragraph]
Aisatsu (挨拶) adalah kata untuk "salam" atau "sapaan" dalam bahasa Jepang. Ini adalah hal pertama yang dipelajari oleh hampir semua orang yang belajar bahasa Jepang — dan memang seharusnya begitu, karena salam adalah pintu masuk ke setiap percakapan.

Selamat pagi:
おはようございます (Ohayō gozaimasu) → formal, untuk orang yang lebih tua atau baru dikenal
おはよう (Ohayō) → kasual, untuk teman atau adik

Selamat siang / sore:
こんにちは (Konnichiwa) → dipakai dari sekitar jam 10 pagi hingga sebelum matahari terbenam
Perhatian: jangan tulis "Konichiwa" — ada dua "n" di tengahnya

Selamat malam:
こんばんは (Konbanwa) → dipakai setelah matahari terbenam
Perhatian: は di sini dibaca "wa", bukan "ha"

Selamat tinggal:
さようなら (Sayōnara) → terkesan final dan formal, seperti "selamat tinggal selamanya"
またね (Mata ne) → kasual, artinya "sampai ketemu lagi"
じゃあね (Jā ne) → sangat kasual, seperti "dah ya"
またあした (Mata ashita) → "sampai besok"

[vocab1]


[FORMAT: Heading 2]
Ungkapan Sopan yang Wajib Hafal

[FORMAT: Paragraph]
Ada beberapa ungkapan yang akan kamu butuhkan hampir setiap hari. Hafalkan ini sekarang:

Terima kasih:
ありがとうございます (Arigatō gozaimasu) → formal
ありがとう (Arigatō) → kasual
どうもありがとうございます (Dōmo arigatō gozaimasu) → sangat formal / penuh rasa terima kasih

Sama-sama:
どういたしまして (Dō itashimashite) → formal, terkesan kaku
いいえ (Iie) → "tidak, tidak apa-apa" — lebih natural dalam percakapan kasual

Maaf / Permisi:
すみません (Sumimasen) → kata paling serbaguna! Bisa untuk minta maaf, memanggil perhatian (seperti "permisi" ke pelayan), atau mengawali pertanyaan
ごめんなさい (Gomen nasai) → minta maaf yang lebih serius, mengakui kesalahan
ごめん (Gomen) → kasual, "sorry"

Tolong:
〜をください (〜 o kudasai) → "tolong berikan..." (untuk minta sesuatu)
おねがいします (Onegai shimasu) → "tolong" secara umum

Tidak apa-apa / OK:
大丈夫です (Daijōbu desu) → "tidak apa-apa" / "saya baik-baik saja"
大丈夫 (Daijōbu) → versi kasualnya

[vocab2]


[FORMAT: Heading 2]
Kata Situasional yang Wajib Tahu

[FORMAT: Paragraph]
Di luar salam formal, ada kata-kata yang akan kamu temui atau butuhkan dalam situasi sehari-hari — terutama saat makan, masuk rumah, dan dalam percakapan kasual:

Sebelum makan:
いただきます (Itadakimasu) → ucapkan sebelum mulai makan, artinya kira-kira "saya humbly menerima makanan ini"
Tidak ada padanan tepat dalam bahasa Indonesia — ini ungkapan rasa syukur dan penghargaan terhadap makanan

Setelah makan:
ごちそうさまでした (Gochisōsama deshita) → "terima kasih atas makanannya", ucapkan setelah selesai makan
ごちそうさま (Gochisōsama) → versi kasualnya

Masuk / keluar rumah:
ただいま (Tadaima) → ucapkan saat pulang ke rumah, artinya "aku pulang"
おかえり / おかえりなさい (Okaeri / Okaerinasai) → respons dari orang di rumah, artinya "selamat datang kembali"
いってきます (Ittekimasu) → ucapkan saat mau pergi, artinya "aku berangkat ya"
いってらっしゃい (Itterasshai) → respons, artinya "hati-hati di jalan"

Di tempat kerja / kelas:
よろしくおねがいします (Yoroshiku onegai shimasu) → ungkapan yang sangat penting, artinya bervariasi: "tolong kerja sama ya", "saya mohon bantuannya", atau "senang berkenalan"
お疲れ様です (Otsukaresama desu) → "kerja keras ya" / "terima kasih atas kerja kerasnya" — dipakai di akhir hari kerja

[vocab3]


[FORMAT: Heading 2]
Perkenalan Diri Pertamamu (自己紹介, Jikoshoukai)

[FORMAT: Paragraph]
Jikoshoukai (自己紹介) artinya perkenalan diri. Ini adalah hal pertama yang biasanya diminta saat kamu bertemu orang Jepang, ikut kelas bahasa Jepang, atau bergabung ke komunitas. Hafalkan template ini — kamu bisa langsung pakai!

Template Perkenalan Diri (5 Kalimat):

はじめまして。
[Nama] と申します。/ [Nama] です。
[Universitas/Organisasi] の学生です。
[Kota/Negara] 出身です。
どうぞよろしくお願いします。

Artinya:
はじめまして → "Salam kenal" / "Senang berkenalan" (dipakai HANYA saat pertama kali ketemu)
[Nama] と申します → "Nama saya [Nama]" (formal)
[Nama] です → "Saya [Nama]" (kasual)
STISの学生です → "Saya mahasiswa STIS"
インドネシア出身です → "Saya berasal dari Indonesia"
どうぞよろしくお願いします → "Senang berkenalan, mohon bimbingannya"

Contoh perkenalan lengkap:
はじめまして。
ハルノと申します。
STISの学生です。
ジャカルタ出身です。
どうぞよろしくお願いします。


[FORMAT: Heading 2]
Kata Tanya Dasar — 5W+1H ala Jepang

[FORMAT: Paragraph]
Sebelum bisa bikin kalimat tanya yang panjang, hafal dulu kata tanya dasarnya:

なに / なん (nani / nan) → apa
だれ (dare) → siapa
どこ (doko) → di mana
いつ (itsu) → kapan
なぜ / どうして (naze / dōshite) → kenapa
どうやって / どのように (dōyatte / dono yō ni) → bagaimana

Contoh penggunaan sederhana:
これは なに？(Kore wa nani?) → Ini apa?
トイレは どこ？(Toire wa doko?) → Toilet di mana?
いくら？(Ikura?) → Berapa harganya?

[FORMAT: Paragraph]
Catatan: kata tanya di atas memang berdiri sendiri dulu. Cara menggabungkannya ke dalam kalimat lengkap akan kita pelajari di artikel berikutnya tentang partikel.
```

### Tabel 1 — [vocab1]

**Judul Tabel:**

```
Salam Berdasarkan Waktu
```

**Baris 1:**
```
Kanji/Karakter : おはようございます
Furigana/Romaji: Ohayō gozaimasu
Arti           : Selamat pagi (formal)
Contoh Kalimat : おはようございます、先生！(Ohayō gozaimasu, sensei!) = Selamat pagi, Pak/Bu Guru!
```

**Baris 2:**
```
Kanji/Karakter : こんにちは
Furigana/Romaji: Konnichiwa
Arti           : Selamat siang / halo
Contoh Kalimat : こんにちは！お元気ですか？(Konnichiwa! Ogenki desu ka?) = Halo! Apa kabar?
```

**Baris 3:**
```
Kanji/Karakter : こんばんは
Furigana/Romaji: Konbanwa
Arti           : Selamat malam
Contoh Kalimat : こんばんは。今日はいい天気でしたね。(Konbanwa. Kyō wa ii tenki deshita ne.) = Selamat malam. Cuacanya bagus ya hari ini.
```

**Baris 4:**
```
Kanji/Karakter : またね
Furigana/Romaji: Mata ne
Arti           : Sampai ketemu lagi (kasual)
Contoh Kalimat : じゃあ、またね！(Jā, mata ne!) = Oke, sampai ketemu lagi!
```

**Baris 5:**
```
Kanji/Karakter : おやすみなさい
Furigana/Romaji: Oyasumi nasai
Arti           : Selamat tidur / malam
Contoh Kalimat : おやすみなさい！(Oyasumi nasai!) = Selamat tidur!
```
