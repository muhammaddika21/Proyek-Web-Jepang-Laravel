Rekomendasi Dashboard Admin

Desain & Arsitektur Sistem Manajemen Konten NihonLearn

🗺️ 1. Struktur Halaman & URL

Dashboard admin akan menggunakan sistem routing yang terproteksi (harus login). Berikut adalah struktur halamannya:

URL Route Fungsi Halaman
/admin Dashboard Utama: Menampilkan statistik, grafik, dan ringkasan aktivitas terbaru.
/admin/articles Daftar Artikel: Tabel semua artikel dengan fitur pencarian, filter, dan pagination.
/admin/articles/create Buat Artikel: Form editor lengkap untuk menulis artikel baru.
/admin/articles/{id}/edit Edit Artikel: Mengubah konten artikel yang sudah ada.
/admin/categories Kategori: Manajemen kategori tulisan (Budaya, Bahasa, dll).
/admin/settings Pengaturan: Konfigurasi dasar website.

💾 4. Rancangan Database (Schema)

Untuk menunjang fitur artikel, diperlukan beberapa tabel utama di Laravel:

Tabel articles

Kolom Tipe Data Keterangan
id BIGINT (PK) Primary Key
title VARCHAR(255) Judul utama artikel
slug VARCHAR(255) URL ramah SEO (contoh: /belajar-hiragana)
content LONGTEXT Isi artikel berformat HTML (dari Rich Editor)
thumbnail VARCHAR(255) Path lokasi gambar sampul
category_id BIGINT (FK) Relasi ke tabel kategori
status ENUM 'published' atau 'draft'
views INTEGER Jumlah dilihat (default: 0)
timestamps DATETIME created_at dan updated_at

Tabel categories

Kolom Tipe Data Keterangan
id BIGINT (PK) Primary Key
name VARCHAR(100) Nama Kategori (Bahasa, Budaya, dll)
slug VARCHAR(100) URL Kategori
color VARCHAR(20) Kode warna hex untuk badge (misal: #448646)

🚀 7. Roadmap Implementasi (Urutan Pengerjaan)

Tahap 1: Fondasi Auth & Database

  • Install Auth Laravel (Breeze/UI) untuk fitur Login Admin.
  • Buat Migration dan Model untuk Article dan Category.
  • Setup layout dasar admin.blade.php (sidebar & header kosong).

Tahap 2: Manajemen Kategori

  • Buat halaman CRUD (Create, Read, Update, Delete) Kategori.
  • Pastikan kategori bisa ditambah dan dihapus.

Tahap 3: Inti Artikel (CRUD)

  • Buat Controller Artikel.
  • Pasang Rich Text Editor (rekomendasi: TinyMCE/Trix).
  • Fungsikan upload gambar Thumbnail.
  • Buat tabel daftar artikel.

Tahap 4: Finalisasi & Integrasi

  • Buat Dashboard Statistik (menghitung jumlah artikel/kategori dari database).
  • Tampilkan data artikel dari database ke halaman publik (Home & Halaman Artikel).
