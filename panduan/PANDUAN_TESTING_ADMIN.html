📋 Panduan Verifikasi Manual NihonLearn

Gunakan daftar ini untuk meninjau dan memastikan semua perubahan terbaru pada Dashboard Admin berjalan dengan baik.

⚠️ Langkah Wajib Sebelum Testing

Buka Command Prompt (CMD), arahkan ke folder project Anda, dan jalankan 2 perintah ini:

php artisan migrate php artisan storage:link

*Juga pastikan php artisan serve dan npm run dev sedang menyala.

Tes 1: Edit Artikel yang Sudah Ada

  1. Buka halaman Kelola Artikel di browser.
  2. Klik tombol ✏️ (Edit) pada salah satu artikel yang sudah ada (misal judul "ddd").
  3. Pengecekan:
    • Judul halaman di pojok kiri atas berubah menjadi "Edit Artikel" / "Edit: ddd".
    • Kolom Judul otomatis terisi teks "ddd".
    • Jika sebelumnya artikel sudah punya kosakata/kuis, tabel tersebut harus sudah terisi data lama.
  4. Ubah judul menjadi "ddd diupdate" lalu klik tombol Simpan Draft di bawah.
  5. Pengecekan:
    • Setelah kembali ke tabel, judul artikel berubah menjadi "ddd diupdate" (artikel ter-update, bukan membuat artikel baru).

Tes 2: Fitur Sorting (Urutkan)

  1. Di halaman Kelola Artikel, perhatikan dropdown filter di sebelah kanan form pencarian.
  2. Ubah filter paling kanan menjadi ↑ Terlama.
  3. Pengecekan:
    • Artikel dengan tanggal paling kuno harus berada di baris paling atas.
  4. Ubah kembali filter tersebut menjadi ↓ Terbaru.
  5. Pengecekan:
    • Artikel yang baru saja diedit ("ddd diupdate") harus berada di bagian atas.

Tes 3: Buat Artikel Baru & Upload Media

  1. Klik tombol biru + Buat Artikel Baru lalu pilih Artikel Bahasa.
  2. Isi Judul, pilih Kategori Bahasa, dan pilih Level Kemahiran (Pemula/Menengah/Mahir).
  3. Scroll ke kotak 📎 Media Tambahan:
    • Pilih tab 🎵 Audio lalu upload sebuah file lagu/audio berformat MP3/WAV.
    • Pilih tab 🖼️ Gambar lalu upload 2 atau 3 gambar bebas.
    • Pilih tab ▶️ YouTube lalu tempelkan link YouTube (misal: https://www.youtube.com/watch?v=dQw4w9WgXcQ).
  4. (Opsional) Isi 1 baris Kosakata dan 1 soal Kuis.
  5. Klik tombol hijau Publish.
  6. Pengecekan:
    • Anda berhasil diarahkan kembali ke tabel dengan pesan sukses hijau di atas tabel.

Tes 4: Tampilan Preview dengan Media

  1. Di tabel, cari artikel bahasa baru yang Anda buat pada Tes 3.
  2. Klik tombol 👁️ (Preview/Logo Mata).
  3. Pengecekan Bagian Konten Kiri:
    • Terdapat bagian baru bernama 📎 Media Tambahan.
    • Galeri Foto: Gambar yang Anda upload muncul dalam bentuk grid. Arahkan mouse ke gambar, nama file aslinya akan muncul.
    • Audio: Muncul pemutar lagu (player) yang bisa ditekan tombol play (▶).
    • YouTube: Muncul bingkai video YouTube yang bisa diputar langsung.
  4. Pengecekan Bagian Sidebar Kanan:
    • Ada kotak bernama Checklist Media.
    • Ceklis warna warni (✓) menyala untuk foto tambahan, audio, dan youtube yang diupload.
    • Nama file asli tercantum di bawah judul checklist tersebut.

Tes 5: Pewarnaan Kuis (Hijau)

  1. Masih di halaman Preview yang sama dengan Tes 4, scroll ke bawah hingga menemukan bagian 🧠 Kuis Review.
  2. Pengecekan:
    • Satu jawaban (yang merupakan kunci jawaban) harus tampil menonjol:
      - Memiliki warna kotak dengan hijau terang.
      - Teks huruf (A/B/C/D) berada dalam bundaran hijau tua.
      - Teks isinya dicetak tebal (bold).
      - Terdapat ikon centang (✓) di pojok kanannya.
    • Sisa tiga pilihan jawaban lainnya tetap berwarna abu-abu polos.
